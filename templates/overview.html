Technology Stack

Random Forest, Decision Tree, and XGBoost are evaluated on the same balanced CIC-IoT23 rows, then XGBoost is used for prediction.

  • Python 3.9+
  • Pandas/NumPy
  • Scikit-learn
  • Flask
  • SMOTE
  • Random Forest
  • Decision Tree
  • XGBoost
Training Pipeline
  1. Load Merged01.csv and Merged02.csv
  2. Keep the same 39 CIC-IoT23 feature columns
  3. Map detailed labels to binary classes: Benign vs Attack
  4. Clean missing and infinite values
  5. Balance the rows with SMOTE and save them as balanced_data.csv
  6. Train RF, DT, and XGBoost on the balanced rows
  7. Use XGBoost for uploads and live monitoring
